# Plataforma-WEB

## PASSO A PASSO RODAR O PROJETO

### 1. ADICIONAR VENV
Use o comando para criar e ativar o ambiente virtual:
`py -m venv venv`
`.\venv\Scripts\activate`

### 2. INSTALAR FLASK
Instale a dependência Flask:
`pip install flask`

### 3. PARA EVENTUAIS PROBLEMAS (PowerShell)
Se houver problemas de permissão na execução, defina a política de execução:
`Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`

### 4. Instalar Requirements (PowerShell)
`pip install -r .\requirements.txt`
